Quick guide

  1. Select a dataset (optional)
  2. Select an audio to play it
    1. Auto-insert images every given number of seconds: choose the "Insert images every" option specifying the time between images.
    2. Select an image from the left panel when you know in which audio position you want to display it while listening to the audio (the first image will be inserted at time 0 at the beginning)
  3. Insert/delete/replace/change position of the images you inserted in the central panel
  4. Copy the result generated in the command line to your terminal (you must have the image and audio files in this directory) and paste the command to execute it and wait for the video to be generated (by default the file “video.mp4” would be generated).
  5. Once ready you can view the video or upload it to your favorite video portal (youtube, vimeo,...).
